Vehicle image sourcing instructions

This file documents the image filenames referenced by `ev.html` and how to supply legally-permitted images for use on the site. Do NOT add manufacturer press images here unless you have written permission that allows commercial/marketing use.

Required filenames (place final web-sized images in this folder):

- `tesla-model-3.jpg`  — Tesla Model 3
- `tesla-model-y.jpg`  — Tesla Model Y
- `tesla-cybertruck.jpg` — Tesla Cybertruck
- `rivian-r1t.jpg`     — Rivian R1T
- `hyundai-ioniq5.jpg` — Hyundai Ioniq 5
- `ford-mach-e.jpg`    — Ford Mustang Mach-E

Guidance:
- Preferred: obtain written permission from the manufacturer for any press/media images used for commercial or marketing purposes. Save permission emails as proof and include the exact attribution string if required.
- Fast fallback: use permissively-licensed stock images (Unsplash, Pexels) or purchase images from paid stock services (Shutterstock, Adobe Stock, Getty). Paid stock gives stronger commercial coverage.
- Image optimization: save final images at the target display size (example grid: 600×400 px) and compress (WebP or optimized JPEG) for web. Keep originals in `assets/originals/`.
- Attribution: if an image license requires attribution, include the exact text in `assets/LICENSES.md` and add a small caption or footer credit on the site.
